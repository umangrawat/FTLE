FTLE CUDA (FTLECUDA)

Computes FTLE on the given patch of the dataset
Computes Finite-Time Lyapunov Exponents on uniform grids.
Property Description Default(s) Restrictions
Input Compute Finite-Time Lyapunov exponent, which characterizes the rate of seperatation of massless tracer particles within a flow.

Accepts input of following types:

  • vtkImageData

The dataset must contain a field array (point) with 3 component(s).

Start time Set initial advection time for the flow map computation 0.0
End time Set final advection time for the flow map computation 15000.0
Maximum Number of Steps The maximum number of steps for the integrator. 10000
Seed grid origin, relative to input Origin of seeding grid. 0 0 0
Seed grid physical extent Size of seeding grid. 2.0 1.0 0.0
Seed grid resolution Resolution of seeding grid. 200.0 100.0 2.0
UseGPU Use GPU for stream/path line integration, not yet implemented. 0

Accepts boolean values (0 or 1).
